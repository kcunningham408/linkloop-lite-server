## Do you or your third-party partners collect data from this app?
YES

- Used for Tracking: NO
- Purpose: App Functionality

### 2. Health & Fitness
- Health: YES (glucose readings — self-entered by user)
- Linked to User Identity: YES
- Used for Tracking: NO
- Purpose: App Functionality

### 3. User Content
- Other User Content: YES (mood entries, notes, chat messages)
- Linked to User Identity: YES
- Used for Tracking: NO
- Purpose: App Functionality

### 4. Identifiers
- User ID: YES (internal database ID)
- Linked to User Identity: YES
- Used for Tracking: NO
- Purpose: App Functionality

### 5. Usage Data
- Product Interaction: NO
- Advertising Data: NO
- Other Usage Data: NO

### 6. Diagnostics
- Crash Data: NO (unless Expo collects this — check your expo config)
- Performance Data: NO
- Other Diagnostic Data: NO
