🎨 What Changed — Visual Overhaul Summary

Glassmorphism Cards — Frosted blur background with subtle gradient borders replacing flat #1C1C1E cards
Glucose Ring — Circular arc gauge around the glucose number showing position in the 40–300 range
SVG Line Chart — Real bezier-curve chart with gradient fill, threshold bands, and time labels (replaces dot grid)
Stat Arcs — Mini circular progress rings for Time in Range, Avg Glucose, Lows, Highs
Floating Tab Bar — Rounded pill shape with blur backdrop, positioned above bottom edge with shadow
Darker Base — Background changed from #111 to #0A0A0F for more contrast with glass cards
